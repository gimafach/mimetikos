## Development Commands

**Start development server:**
```bash
npm run dev
# or
bun dev
```
Server runs on port 8080 with HMR enabled.

**Build:**
```bash
npm run build          # Production build
```

**Linting:**
```bash
npm run lint           # Run ESLint
npm run format         # Format with Prettier
```

**Preview:**
```bash
npm run preview        # Preview production build
```

## Project Architecture

React + TypeScript single-page application built with Vite and TanStack Router.

**Technology Stack:**
- **Frontend Framework**: React 19 with TypeScript
- **Build Tool**: Vite with React plugin
- **Styling**: Tailwind CSS v4 with shadcn/ui component library
- **State Management**: TanStack Query for server state
- **Routing**: TanStack Router (file-based, SPA mode)
- **Server (prod)**: Express.js serving `dist/`
- **Process Manager**: PM2 (`ecosystem.config.cjs`), port 4002
- **Package Manager**: bun preferred (bun.lock present)

**Project Structure:**
- `src/routes/` - Page routes (`__root.tsx`, `index.tsx`, `en.tsx`)
- `src/components/ui/` - Reusable UI components from shadcn/ui
- `src/hooks/` - Custom React hooks
- `src/lib/` - Utility functions
- `src/assets/` - Images and static assets

**Key Features:**
- **Two-language site**: Italian (`/`) and English (`/en`)
- **File-based routing**: TanStack Router plugin auto-generates `src/routeTree.gen.ts`
- **Component System**: shadcn/ui with Radix UI primitives
- **Responsive Design**: Tailwind CSS v4

**Import Aliases:**
- `@/*` maps to `src/*` (configured in vite.config.ts and tsconfig paths)

**Development Notes:**
- Uses ES modules (`"type": "module"`)
- `src/routeTree.gen.ts` is auto-generated by `@tanstack/router-plugin` — do not edit manually
- Production build output goes to `dist/`, served by `server.js`
- PM2 process name: `mimetikosit`

**Adding New Routes:**
Add a new file under `src/routes/`. TanStack Router plugin auto-generates `routeTree.gen.ts` at build/dev time.
